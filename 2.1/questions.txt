Какие концепции лежат в основе стандартной библиотеки?
Стандартная библиотека С++ содержит набор универсальных классов и интерфейсов, значительно расширяющих ядро языка С++.
В основу разработки библиотеки положена парадигма обобщенного программирования.

Зачем в проектах используются системы контроля версий?
Чтобы была возможность сделать откат назад, посмотреть, кто и как изменял проект

Из каких основных действий состоит взаимодействие с Git?
git status, git add, git commit, git checkout, git clone, git push, git fetch, git pull, git merge...

Когда следует создавать отдельные ветки для разработки?
В master основные и (хорошо бы) рабочие версии проекта. 
Ветки следует делать в случае коллективной работы, при наличии большого количества подзадач

Какие основные элементы содержатся в библиотеке chrono?
С++11:
Time Point — задается точкой отсчета (так называемой «эпохой») и сопутствующим временным интервалом.
Time duration — разница между двумя моментами времени. Определяется числом «тиков» времени.
Clock — состоят из точки отсчета (эпохи) и «тика» времени. Эта информация позволяет рассчитать текущее время.
В С++20 добавлены:
Time of day — это временной интервал, ведущий свой отсчет с полуночи, представляемый в виде часов, минут, секунд и долей секунды.
Calendar — обозначает различные календарные даты, такие как: год, месяц, день недели или n-й день недели.
Time-zone — представляет собой время, характерное для определенной географической области.
