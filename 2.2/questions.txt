В каких ситуациях применяются типы std::pair, std::tuple?
pair - для хранения 2 значений произвольных типов, tuple - n значений
Когда следует использовать std::array?
Когда нужен массив фиксированного размера

Когда следует использовать std::vector?
Когда нужен динамический массив - почти всегда

Когда следует использовать std::deque?
Двустороняя очередь, подходит для оперций вставок в начало и конец за О(1), так как имеет страничную структуру

Когда следует использовать std::list?
Двусвязный список, подходит для вставки и удаления элементов в произвольное место за О(1)

Когда следует использовать std::forward_list?
Используется как деталь, является односвязным списком, одно направление, максимально экономит память

Какие адаптеры контейнеров есть в стандартной библиотеке?
стек - LIFO
очередь - FIFO
очередь с приоритетами - FIFO+sort

Когда следует использовать circilar buffer из Boost?
Когда история фиксированной длины и постоянно поступают новые данные

Почему контейнер circilar buffer из Boost не может войти в стандартную библиотеку?
Потому что возможна ситуация, когда begin>end

Какие типы данных есть для работы с многомерными массивами?
boost::multiarray, std::valarray, std::vector
